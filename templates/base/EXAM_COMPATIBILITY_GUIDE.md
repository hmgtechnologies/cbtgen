# Exam Compatibility Guide — HMG Academy CBT Pro v4

HMG Academy CBT Pro is designed to support many school-ecosystem assessment types without paid APIs.

## Supported exam contexts

- Nursery/Primary diagnostic quizzes
- Common entrance examinations
- Junior secondary tests and BECE practice
- Senior secondary class tests and terminal exams
- WAEC/NECO/NABTEB-style practice
- UTME/JAMB and Post-UTME practice
- Scholarship/admission screening
- External candidate entrance tests
- Training and certification exams
- Staff recruitment aptitude tests
- Coding/algorithm assessments
- Science practical/theory hybrid assessments
- School club/competition quizzes

## Useful features by scenario

### Common entrance / external students

- Open mode: students enter name and class/category.
- Access code/link sharing.
- Instant teacher-side printing of result slips.
- Instant teacher-side certificate printing.
- Bulk print filtered results/certificates.

### Registered internal exams

- Registered-student mode with student ID verification.
- Student roster CSV import.
- Attempt limit.
- Result exports.
- Analytics and item analysis.

### High-integrity exams

- Tab/app switch logging.
- Copy/paste/right-click controls.
- Fullscreen enforcement.
- Optional camera proctoring.
- Optional audio spike monitoring.
- Violation logs.

### STEM exams

- Math/Science keyboard available for all active exams.
- Numeric with tolerance.
- Multi-part numeric.
- Matching, ordering, cloze and matrix questions.
- Scientific calculator.

### Certification/training exams

- Certificate verification code.
- Print certificates.
- Export results CSV.
- Question/answer packet archive.

## Recommended setup for common entrance

1. Create exam in open mode.
2. Keep attempt limit at 1.
3. Enable result release if candidates should see results instantly.
4. Add school-specific instructions.
5. After candidates submit, filter by exam in Teacher Results.
6. Use `Print Listed Results` or `Print Listed Certificates`.
7. Export CSV for admission office record.
