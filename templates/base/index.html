How It Works

01
📄

Upload Questions

Teachers prepare questions in a CSV, Excel, or type them directly. The system parses everything automatically with support for 17 question types.

02
⚙️

Configure the Exam

Set duration, question count, randomisation, passing score, attempt limits, and exam mode. Full editing available at any time.

03
📲

Share the Code

Share the exam code or WhatsApp link. Students enter it on the student portal — no registration needed, works on any device.

04
📊

Analyse Results

Results appear instantly with score distributions, pass/fail ratios, average time-per-question, and CSV export for further analysis.

Everything a Classroom Needs

No paid add-ons. No hidden limits. Every feature ships free.

🔀

Randomised Questions & Options

Questions and answer options are shuffled per student, making it harder to copy and improving exam integrity.

🚩

Question Flagging

Students can flag uncertain questions and jump back to them before submitting — reducing careless mistakes.

⏱️

Countdown Timer

A live countdown timer with colour warnings keeps students aware of remaining time throughout the exam.

🏅

Result Certificate

Students receive a shareable certificate immediately after submission showing their score and performance grade.

💬

Teacher Explanations

Teachers can attach explanations to each question, which students see during their result review.

🔒

One Submission Only

The system prevents students from submitting multiple times, ensuring fair and clean result records.

📈

Analytics & Charts

Teachers see score distributions, pass/fail ratios, average time-per-question, and more on built-in charts.

📲

WhatsApp Sharing

Generate a direct WhatsApp link from any exam — students can join with one tap from the shared message.

✏️

Full Exam Editing

Edit every aspect of any exam — subject, class, term, topic, duration, pass mark, question count, exam mode. Add or remove questions at any time.

📦

Exam Package Export/Import

Export entire exams as JSON packages for backup, sharing, or re-use across different deployments and terms.

✅

Deployment Validator

A free browser-only readiness checker verifies required files, PWA assets, Supabase configuration, and security basics.

📖

Detailed Feature Guide

A built-in system guide explains every portal, feature, question type, free tool, and deployment step for school administrators.

Download / Install on Any Device

Use HMG Academy CBT Pro like an app without visiting an app store.

🤖

Android phones/tablets

Open the site in Chrome, tap the menu ⋮, then choose Install app or Add to Home screen.

🍎

iPhone / iPad

Open in Safari, tap Share, then choose Add to Home Screen. The CBT icon will appear like a normal app.

💻

Windows / macOS / Chromebook

Open in Chrome or Edge and click the install icon in the address bar, or use browser menu → Install HMG CBT.
